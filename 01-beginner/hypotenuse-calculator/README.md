# Hypotenuse Calculator

Find the length of side c using java

## 📋 Features
- Takes inputs for side a and side b
- Prints hypotenuse length
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter length of side a
3. Enter length of side b

## 💡 Concepts Used
- Math functions

## 📸 Sample Output
====HYPOTENUSE CALCULATOR====

Enter Length of Side A : 3
Enter Length of Side B : 4

The Length of the Hypotenuse [c] : 5.0
