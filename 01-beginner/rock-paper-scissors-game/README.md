# Rock Paper Scissors Game ❓

A rock paper scissors game written in Java.

## 📋 Features
- Provids 3 choices
- Play with computer
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter choice

## 💡 Concepts Used
- Conditions
- User input validation
- Random numbers
- Enhanced switches
