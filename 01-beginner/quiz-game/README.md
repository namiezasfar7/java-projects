# Quiz Game ❓

A mini quiz game written in Java.

## 📋 Features
- Provids 3 questions
- Tracks score
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter answer

## 💡 Concepts Used
- Conditions
- User input validation
