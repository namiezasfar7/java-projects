# Number Guessing Game 🔢

A number guessing game that generates a random number between 1 and 100

## 📋 Features
- Tracks number of tries
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter number

## 💡 Concepts Used
- Loops and conditions
- User input validation
- Random numbers
