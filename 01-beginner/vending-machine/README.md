# Vending Machine 📠

A vending machine simulated in java

## 📋 Features
- Gets user input for choice and quantity
- Calculates total
- Applies 10% discount if total is > Rs. 1000
- Displays final amount
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter choice
3. Enter quantity

## 💡 Concepts Used
- Loops and conditions
- Enhanced Switches
- User input validation
