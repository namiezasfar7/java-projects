# ATM PIN System 🏧

An ATM PIN System written in Java.

## 📋 Features
- Checks validation 3 times
- Locks user if user attempts 3 times
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter PIN

## 💡 Concepts Used
- Loops and conditions
- User input validation

## 📸 Sample Output
Enter PIN : 1234
Access Granted!
