# Simple Fizz Buzz

Just a simple repetition to check multiples of 3 and 5

## 📋 Features
- Prints FIZZ for multiples of 3
- Prints BUZZ for multiples of 5
- Prints FIZZBUZZ for multiples of 3 and 5
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
