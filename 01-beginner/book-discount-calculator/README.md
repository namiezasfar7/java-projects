# 📚 Book Discount Calculator

A simple Java-based Book Discount Calculator that demonstrates the concepts of **inheritance**, **constructors**, **method creation**, and **basic object-oriented programming (OOP)**. The program models books and novels, calculates discounted prices for novels, and displays the resulting information.

---

## 📋 Features

- Models books using a base `Book` class
- Extends the `Book` class using a `Novel` subclass
- Calculates discounted prices for novels
- Demonstrates constructor inheritance using `super()`
- Displays book details and discounted prices
- Console-based Java application

### Compile

```bash

```

### Run

```bash
java Main
```

---

## 📖 Usage

1. Compile and run the program.
2. The application creates two `Novel` objects.
3. Each novel inherits its basic information from the `Book` class.
4. The program calculates the discounted price using the specified discount rate.
5. The book details and discounted prices are displayed.
