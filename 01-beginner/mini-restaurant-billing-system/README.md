# Mini Restaurant Billing System 😋

A mini restaurant billing system that tracks total number of items and total bill

## 📋 Features
- Calculates total bill
- Stores number of items bought
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Check menu
3. Enter choice
4. Enter number of items

## 💡 Concepts Used
- Loops and conditions
- User input validation
