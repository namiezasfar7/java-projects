# Mad Libs Game 🎮

A fun mad libs game created in java

## 📋 Features
- Takes 5 inputs
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter adjectives, nouns and verbs

## 💡 Concepts Used
- User input validation

## 📸 Sample Output
Enter an adjectice (description) : large
Enter a noun (animal or person) : lion
Enter an adjectice (description) : fierce
Enter a verb ending with -ing (action) : roaring
Enter an adjectice (description) : bamboozled

Today I went to a large zoo.
In an exhibit, I saw a lion.
lion was fierce and roaring!
