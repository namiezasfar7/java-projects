# Student Result System 🧑‍🎓

A student result system that calculates total, average and finds grade

## 📋 Features
- Gets user input for maths, science and english
- Calculates total and average
- Finds grade
- Displays final result
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter marks

## 💡 Concepts Used
- Loops and conditions
- User input validation
