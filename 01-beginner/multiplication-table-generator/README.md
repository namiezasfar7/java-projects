# ✖️ Multiplication Table Generator

A simple Java-based Multiplication Table Generator that allows users to generate the multiplication table of any number for a specified number of multiples. This project demonstrates the use of user input, loops, arithmetic operations, and formatted output in Java.

---

## 📋 Features

- Accepts any integer from the user
- Allows the user to specify how many multiples to generate
- Displays a neatly formatted multiplication table
- Uses a `for` loop for table generation
- Simple console-based interface

---

## 🛠️ Technologies Used

- Java
- Scanner Class
- Variables
- Arithmetic Operators
- Loops (`for`)
- Formatted Output (`printf`)

## ▶️ How to Compile and Run

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
```

---

## 📖 Usage

1. Run the program.
2. Enter the number whose multiplication table you want to generate.
3. Enter how many multiples should be displayed.
4. The program will generate and display the multiplication table.
