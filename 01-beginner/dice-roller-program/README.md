# Dice Roller Program 🎲

A dice roller program written in Java.

## 📋 Features
- Asks user for input
- Rolls random numbers
- Prints total
- Contains ASCII art

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter number of die

## 💡 Concepts Used
- Conditions
- Methods
- Enhanced switches
- User input validation
