# Employee Payroll System 👔

An employee payroll system that calculates NET salary.

## 📋 Features
- Gets inputs for name, salary and work hours
- Gives NET salary

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Enter name
3. Enter salary
4. Enter work hours

## 💡 Concepts Used
- Classes and objects
- Setters and getters
- Initialization

## 📸 Sample Output
Enter Employee Name     : Namiez
Enter Employee Salary   : Rs. 200000
Enter Hours Worked      : 50
