# Simple Banking System 🏦

A simple banking system that allows to withdraw and deposit cash

## 📋 Features
- Stores balance, and allows deposit and withdraw
- Checks user input validation
- Simple logic

## 🛠️ How to Run
```bash
javac Main.java
java Main
```

## 📖 How to Use
1. Run the program
2. Check menu and enter choice

## 💡 Concepts Used
- Loops and conditions
- User input validation

## 📸 Sample Output
1. Check Balance
2. Deposit
3. Withdraw
4. Exit
